# Simple Taskboard

A simple task management application for organizing tasks using a Kanban-style board.

🌐 Live Demo: https://monkfish-app-yf49j.ondigitalocean.app/

> User authentication and additional user features are currently under development.

## Tech Stack

| Layer | Stack |
|-------|-------|
| Frontend | React, Vite, Tailwind CSS |
| Backend | ASP.NET 8 Web API |
| Database | PostgreSQL with Entity Framework Core |

## Project Structure

This project uses a monorepo architecture to manage both frontend and backend in a single repository.
